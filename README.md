# LibreTest. An open testing platform.

**Your test. Your medium. Your rules. Your way. That's LibreTest.**

LibreTest is an exam making and taking platform that is free and open source, under the Apache 2.0 License.

This README is valid for version 0.1.x of LibreTest.

## Features

- JSON exam making
- Multiple choice exams
- Timed testing
- Start button
- Auto-submit on timeout
- Roboto font everywhere
- More to come soon...

## Getting Started

```bash
cd frontend
npm install
npm run dev
