# LibreTest. An open testing platform.

**Your test. Your medium. Your rules. Your way. That's LibreTest.**

LibreTest is an exam making and taking platform that is free and open source, under the Apache 2.0 License.

This README is valid for version 0.2.x of LibreTest.

## Disclaimer

LibreTest is an independent open source project. It is not affiliated with:

- The Document Foundation (LibreOffice)
- The Linux Foundation
- Open Assessment Technologies (TAO)
- Moodle
- ACT, Inc.
- College Board
- International Baccalaureate Organisation (IBO)
- Any other foundation, exam board, or organization

LibreTest is its own entity. The "Libre" prefix reflects the philosophy of free as in freedom, not affiliation with any organisation.

## Features

- JSON exam making
- Multiple choice exams
- Timed testing
- Start button
- Auto-submit on timeout
- Roboto font everywhere
- More to come soon...
